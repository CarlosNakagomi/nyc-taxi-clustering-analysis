# NYC taxi trip clustering (methodology study, Python)

This is a **Python script–based** clustering study using **NYC TLC–style** yellow taxi trip data. It is **not** a web app, **not** a Streamlit dashboard, and **not** a production pipeline—only a **reproducible methodology** workflow: loading / cleaning, **scaling**, **PCA**, **clustering tendency (Hopkins)**, **elbow** and **silhouette** for *k*, **K-Means (k = 4)**, **validation** metrics from **scikit-learn**, and figures including **time**, **fare**, a **Matplotlib** hotspot scatter, and a **Plotly** map with an **OpenStreetMap** basemap and **cluster centroids**.

**Raw** NYC TLC trip files are **not** in the repository (they can be very large). You may either:

- Place the public [TLC](https://www.nyc.gov/site/tlc/about/tlc-trip-record-data.page) **yellow** taxi CSVs for **2016-01, 2016-02, 2016-03** under `data/raw/` (see filenames in `run_analysis.py`), or  
- Use an existing `data/processed/yellow_sample_export.csv`, or  
- Run with **no data** at all: the script will **synthesize a small, realistic demo cohort** in NYC bounds, save it to `data/processed/yellow_sample_export.csv`, and complete the run **end-to-end** for **reproducibility**.

PAM / K-Medoids comparisons (sometimes used in the **R** version) are **not** part of the default **Python** workflow so the project does **not** need **scikit-learn-extra** or a **C++** toolchain.

## Run

```bash
python -m venv .venv
# .venv\Scripts\activate   # Windows
# source .venv/bin/activate  # macOS / Linux
pip install -r requirements.txt
python run_analysis.py
```

**Outputs**

- `outputs/figures/` — PNGs (and `nyc_cluster_centroid_map.html` for the interactive **OSM** map; `nyc_cluster_centroid_map.png` is written **if** `kaleido` is installed)  
- `outputs/tables/` — CSV/JSON text summaries (overview, validation, cluster means, etc.)

## What gets produced (high level)

- **Dataset** overview and preprocessing note  
- **StandardScaler** on numeric features, **PCA** + scree, **Hopkins** statistic (clustering tendency)  
- **Elbow** and **Silhouette** sweeps (k = 2…10) on a subsample  
- **Ward** hierarchical **silhouette** at *k* = 4 (reference)  
- **K-Means** *k* = 4: **Silhouette**, **Davies–Bouldin**, **Calinski–Harabasz**, and an extra **Dunn** index in code  
- **Cluster** labels: **Standard Trips**, **Premium Trips**, **Corporate/Low Tip**, **Group Rides** (data-driven mapping from per-cluster means)  
- **Fare** 1-D K-Means bands, **hour** histogram, **OSM** map (Plotly) with **pickups** and **centroid** markers and “~4 mile” **reference** circles, plus a static **Matplotlib** hotspot scatter (no basemap)
